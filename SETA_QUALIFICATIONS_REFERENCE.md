## NQF Level Summary:
- **NQF Level 1**: Basic education, foundational skills
- **NQF Level 2-3**: Intermediate skills, technical certificates
- **NQF Level 4**: Senior certificates, technical diplomas
- **NQF Level 5**: Diplomas, advanced certificates
- **NQF Level 6**: Advanced diplomas, postgraduate certificates
- **NQF Level 7-8**: Bachelor's degrees, advanced diplomas
- **NQF Level 9-10**: Postgraduate qualifications
- **NQF Level 10**: Master's degrees
- **NQF Level 9-10**: Doctoral degrees, professional qualifications

This reference provides comprehensive qualification mappings for implementing dynamic qualification selection based on SETA type selection.
